# Changelog

All notable changes to this repository will be documented in this file.

## 2026-07-04

### Added

- Initial public repository structure
- `rules/custom-direct.txt` for self-maintained direct rules
- `scripts/build_overlay.py` to generate the whitelist overlay
- `scripts/merge_subscription.py` to merge rules into an existing Mihomo subscription
- `scripts/sync_loyalsoldier.ps1` to fetch upstream whitelist rule sets
- `THIRD_PARTY.md` to document upstream rule sources
- `example/base-subscription.yaml` as a merge input example

- Repository branding renamed to `Clash 白名单规则（自用）`
- GitHub repository renamed to `clash-whitelist-self-use`
